# Changelog

All notable changes to the "Copilot Instruction Resolver" extension.

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- Tree view showing all discovered instruction files
- Instruction hierarchy visualization (web and markdown)
- Status bar integration showing active instruction count
- Support for walking up parent directories to find instructions
- Merging strategies: hierarchical and concatenate
- Workspace-specific instruction overrides
- Quick navigation to any instruction file
- File watcher for automatic updates
- Configuration options for customization
- Support for `.github/copilot-instructions.md` files
